Advanced Algorithms

Medical-grade analysis in a budget device

HRV Analysis

RMSSD: Root Mean Square of Successive Differences
SDNN: Standard Deviation of NN intervals
pNN50: Percentage of successive RR > 50ms
Reference: Normal RMSSD = 20-80ms

AFib Detection

RR interval irregularity analysis using Coefficient of Variation (CV).
Sensitivity: 95%
Specificity: 90%
CV Threshold: > 10%

BP Estimation

PPG waveform analysis using Pulse Transit Time (PTT) method.
Combines HRV, Perfusion Index, and HR for estimation.
Accuracy: +/-10 mmHg

Fall Detection

MPU6050 acceleration threshold: > 2.5g
Free-fall detection: < 0.5g for > 100ms
Orientation change: > 90 degrees
Alert: Immediate SMS + Vibration
