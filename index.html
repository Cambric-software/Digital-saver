✨ Features

❤️ Heart Rate

Real-time monitoring + HRV analysis with RMSSD, SDNN, pNN50

💓 AFib Detection

Atrial fibrillation screening algorithm

🩸 Blood Pressure

PPG waveform analysis + vascular age estimation

⚠️ Fall Detection

Accelerometer-based loss of consciousness detection

📍 Emergency Alerts

SMS/Call with GPS location to contacts

🌐 10 Languages

English, Arabic (RTL), Spanish, French, German, Chinese, Japanese, Russian, Portuguese, Hindi

😴 Sleep Analysis

Deep/Light/REM sleep stages

📊 Health Score

Comprehensive 0-100 health assessment

🚀 Run the App

# Clone the repo
git clone https://github.com/asserkdev/Digital-saver.git
cd Digital-saver/app

# Install dependencies
flutter pub get

# Run on device
flutter run

🔧 DIY Smartwatch

The app connects to a handmade ESP32 smartwatch. See hardware/ folder for:

  • Firmware (ESP32 Arduino code)
  • Circuit schematics
  • Assembly guide
  • Component list (~2,810 EGP)

This is a wellness/screening tool and is NOT a medical device.
